# Student Marks Dataset — Pandas Analysis 📊

A data analysis project exploring student marks using **Python** and **Pandas**. The project loads a raw mark sheet, cleans it, and extracts insights such as averages, top performers, and pass/fail trends.

---

## 📌 Project Overview
This project uses a student mark sheet dataset to practice real-world data analysis: loading data, cleaning it, and drawing simple insights with Pandas. It's part of my journey toward becoming a Data Analyst.
